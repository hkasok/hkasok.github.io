Softwares & Tools

I actively develop and customize computational tools to support my research in urban heat island modeling, CFD simulation, and environmental data analysis. My focus is on creating flexible and efficient workflows that couple physical modeling with data-driven techniques.

1. OpenFOAM Customization

I maintain a modified branch of OpenFOAM that integrates immersed boundary formulations, radiative heat transfer, and vegetation parameterization. Key developments include:

  • Enhanced fvDOM solar radiation model with spectral bands and sky-view factor support.
  • Custom solvers for coupled heat, momentum, and moisture transport in urban microclimates.
  • Implementation of immersed boundary (IB) methods for complex tree and building geometries.
  • CFD post-processing utilities for temperature, turbulence, and energy flux diagnostics.

2. Python & MATLAB Tools

I develop Python and MATLAB scripts for sensor data processing, visualization, and model validation. These tools streamline the handling of large field datasets and simulation outputs.

  • Automated data cleaning and merging for multi-sensor Onset HOBO networks.
  • Time-synchronized analysis of temperature, humidity, radiation, and soil moisture.
  • 3D and time–height plots for CFD–experiment comparison.
  • Machine learning integration for predicting microclimate variables and model calibration.

3. GitHub & Open Source Contributions

Selected tools and solvers will be made available through my GitHub profile, including OpenFOAM case templates, radiation model add-ons, and post-processing scripts.

4. Software Ecosystem

  • CFD: OpenFOAM (v2312–v2406, v2206-IBM) with custom solvers
  • Pre/Post-processing: ParaView, Blender, FreeCAD, Tecplot
  • Data Analysis: MATLAB, Python (NumPy, Pandas, Matplotlib, SciPy)
  • Machine Learning: TensorFlow, PyTorch, Scikit-learn
  • Version Control: GitHub, Bitbucket
